HELP: to get help type help or use help <command>

Available commands:

    - hello
    - name
    - version
    - exit
    - echo [string]
    - hash <string>
    - help <string>
    - set <string> <int> (see Global variables for more detail)
    - cat <file>
    - view <string>

Global variables:
    - DEBUG (int)
    - TRANSPARENCY (int)

Const variables:
    - VERSION (string)
